# odin-recipes

Ce projet est le premier que je réalise en suivant la formation du site The Odin Project (theodinproject.com). Je dois y mettre à l'épreuve les compétences que j'ai acquises en HTML durant la formation, tout en mettant en place un espace de travail sur GitHub via Git (lignes de commande).

Je suis tout ouvert aux feedbacks que vous pourriez m'apporter !

La formation se limite pour l'instant au HTML, mais le projet sera continué pendant la formation avec l'acquisition de nouvelles compétences sur d'autres langages de programmation (prochainement CSS).
